# [Fenster](https://github.com/zserge/fenster)
The most minimal cross-platform GUI library.
Requires only Xlibs (libX11, libcxb, libXau, libXdmcp) on Linux. No additional requirements for Windows (gdi32) or MAC (Cocoa)
